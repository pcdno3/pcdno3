(단, 화면 출력 명령문을 3번 써서 완성하시오.)

	public static void main(String[]args){
	
	String year="2010년";
	String month="3월";
	String day="11일";
	
	System.out.print("[연도 - "+year+"]");
	System.out.print(" [월 - "+month+"]");
	System.out.print(" [일 - "+day+"]");
	}
}
